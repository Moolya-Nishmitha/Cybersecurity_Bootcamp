# Simple Python Calculator

A beginner-friendly command-line calculator built with Python. This project performs the four basic arithmetic operations based on user input.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Prevents division by zero
* Handles invalid operation input

## Technologies Used

* Python 3

## How to Run

1. Clone this repository.
2. Navigate to the project folder:

```bash
cd calculator
```

3. Run the program:

```bash
python calc.py
```

If `python` doesn't work on your system, try:

```bash
py calc.py
```

## Example

```text
Simple Calculator <3

Enter first number: 12
Enter second number: 4
Choose operation (+, -, *, /): *

48.0
```
