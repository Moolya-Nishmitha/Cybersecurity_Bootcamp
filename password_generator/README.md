# 🔐 Password Generator

A simple Python password generator that creates random passwords based on a user-defined length. Users can also choose whether to include special characters in the generated password.

## 🚀 Features

* Choose the password length.
* Includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
* Optional special characters.
* Generates a random password each time.

## 🛠️ Technologies Used

* Python 3
* `random` module
* `string` module

## 📂 Project Structure

```text
password_generator/
└── randpass.py
```

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run:

```bash
python randpass.py
```

4. Enter the desired password length.
5. Choose whether to include special characters (`y` or `n`).
6. Your password will be displayed in the terminal.

## 📸 Example

```text
Enter password length: 12
Include special characters? (y/n): y

Generated Password:
T@9m#Q2!xL7$
```
